# Product Guide - Short Creator

## 1. Initial Concept
한국 20-30대를 타겟으로 하는 "Would You Rather" 쇼츠 영상을 자동으로 생성하고 업로드하여 높은 조회수와 참여율을 달성하는 자동화 시스템입니다.

## 2. Target Audience
- **한국 20-30대 쇼츠 시청자**: 일상적이고 공감 가능한 선택지에 흥미를 느끼는 주 타겟층
- **유튜브 채널 운영자 및 콘텐츠 크리에이터**: 콘텐츠 제작 자동화를 통해 채널 성장을 가속화하려는 사용자

## 3. Core Goals
- **바이럴 콘텐츠 생성**: 높은 조회수와 참여율을 유도하는 쇼츠 영상 자동 생성
- **100% 자동화 파이프라인**: 질문 생성부터 이미지 소싱, 영상 렌더링, YouTube 업로드 및 분석까지 전 과정 자동화
- **효율적인 제작**: 제작 시간(10개 영상 / 10분)과 비용을 획기적으로 절감

## 4. Key Features
- **Gemini 기반 질문 생성**: Google Gemini API를 활용하여 한국 정서에 맞는 재미있고 현실적인 질문 자동 생성
- **이미지 자동 소싱**: Pexels API를 사용하여 질문 키워드에 적합한 고품질 스톡 이미지 자동 검색 및 다운로드
- **한국어 TTS 합성**: Typecast API의 인기 캐릭터(박창수, 개나리 등)를 활용한 친근한 음성 생성
- **고속 로컬 렌더링**: Canvas로 화면을 구성하고 FFmpeg를 사용하여 1080x1920 세로형 쇼츠 영상 제작
- **업로드 및 분석**: YouTube Shorts 자동 업로드 및 성과 분석 데이터 수집

## 5. Design Principles
- **모듈화 및 확장성**: 인터페이스 기반 설계(`IQuestionGenerator`, `ITTSProvider` 등)를 통해 특정 서비스(예: TTS, LLM)를 쉽게 교체 가능
- **비용 최적화**: 영상당 단가를 최소화하여 대량 생산이 가능한 지속 가능한 구조 유지
- **로컬 처리 효율성**: 클라우드 의존도를 낮추고 로컬 자원을 활용하여 고속 렌더링 구현
