# Tech Stack - Short Creator

## 1. Core Environment
- **Runtime**: Node.js (v20.x+)
- **Language**: TypeScript
- **Package Manager**: npm

## 2. External APIs
- **LLM**: Google Gemini API (Content & Question Generation)
- **TTS**: Typecast API (Korean Character Voices: 박창수, 개나리 등)
- **Images**: Pexels API (Stock image sourcing)

## 3. Media Processing
- **Image Composition**: Node-canvas (Frame layout, Text rendering, Image manipulation)
- **Video Rendering**: FFmpeg (Merging frames and audio, H.264 encoding)

## 4. Assets & UI
- **Typography**: Pretendard (Optimized for Korean readability on mobile)
- **Design Assets**: Programmatically generated gradients and overlays

## 5. Development Tools
- **Linter/Formatter**: ESLint, Prettier
- **Build Tool**: tsc (TypeScript Compiler)
